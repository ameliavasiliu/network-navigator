The Tasks tab must be completely restructured.

Tasks Tab Layout

Left: Daily Task

Right: Weekly Task

No long vertical task list.
No roadmap duplication.

Task Refresh Rules

Weekly Task refreshes Monday at 12:00 AM

Task Philosophy

Tasks are coaching modules, not checklists.

Think:

Duolingo-style progression

Small daily wins

One deeper weekly effort

Task Interaction Flow

When user clicks a task card:

Navigate to a dedicated task screen

Guide the user through 3–4 steps max

Each step must require real input or action

Task Step Requirements

Each task includes:

Objective

Why this matters

Optional International Consideration (collapsible)

Step-by-step guidance (extremely detailed, beginner-friendly)

AI assistance if relevant

Completion validation

VALID TASK COMPLETION (CRITICAL)

A task CANNOT be completed by typing random text.

Completion must involve ONE of:

Uploading a file (resume, notes)

Answering guided questions (quiz-style, Duolingo-like)

Adding/updating Contacts

Adding/updating Companies

Logging outreach or follow-ups

Examples:

Networking Daily Task

Add 2 new contacts to Contacts tab

OR update status of existing contacts

OR log a follow-up message

Weekly Networking Task

Identify 5 alumni via LinkedIn

Add them to Contacts tab

Company Research Task

Learning Task
